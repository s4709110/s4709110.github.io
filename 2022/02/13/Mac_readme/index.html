USB HDMI 擷取卡設定教學 Mac os

  • 蝦皮 e+宜家數位 連結

測試可擷取裝置

  • Switch(原廠底座)
  • 安博盒子
  • 小米盒子
  • Tivo 電視棒
  • MOD(有HDCP保護)

Mac os 設定步驟

  • 1.下載OBS 連結
  • 2.打開 OBS 後，要先設定遊戲畫面出來，在下方的來源中，
    按下 ＋ 新增「視訊擷取裝置」。

  • 3.直接按「確定」。


  • 4.設定完成

音訊設定

  • 直接按「確定」

  • 「齒輪/高級音訊屬性」

  • 麥克風/AUX ->選「僅監聽(輸出靜音) 」
